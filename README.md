# Ajinkya Sultane — Product Engineer Portfolio

Premium dark-mode portfolio built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Three.js.

## Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion**
- **Three.js** + React Three Fiber
- **React Icons**

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm start
```

## Resume

Add your PDF to `public/resume.pdf` for the Download Resume button.

## Legacy site

The previous static HTML portfolio is preserved in `legacy/`.
